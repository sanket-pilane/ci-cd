Jenkins and Docker Information

Jenkins

Jenkins is an open-source automation server that enables developers to reliably build, test, and deploy their software.

Common Jenkins Commands (on Ubuntu):

sudo systemctl start jenkins

Starts the Jenkins service.

sudo systemctl stop jenkins

Stops the Jenkins service.

sudo systemctl restart jenkins

Restarts the Jenkins service.

sudo systemctl status jenkins

Checks the current status of the Jenkins service.

sudo journalctl -u jenkins.service

View the logs for the Jenkins service.

sudo cat /var/log/jenkins/jenkins.log

View the main Jenkins log file (if it exists and Jenkins starts).

Docker

Docker is a platform for developing, shipping, and running applications inside containers. Containers are lightweight, portable, and self-sufficient units.

Common Docker Commands:

Container Management

docker run [OPTIONS] IMAGE [COMMAND] [ARG...]

Runs a Docker container from an image.

Example: docker run -d -p 80:80 nginx (runs an Nginx container in detached mode, mapping port 80 of the host to port 80 of the container).

docker stop CONTAINER_ID

Stops a running Docker container.

You can get the CONTAINER_ID using docker ps.

docker start CONTAINER_ID

Starts a stopped Docker container.

docker restart CONTAINER_ID

Restarts a Docker container.

docker rm CONTAINER_ID

Removes a stopped Docker container.

docker ps

Lists running Docker containers.

docker ps -a

Lists all Docker containers (running and stopped).

Image Management

docker build -t IMAGE_NAME .

Builds a Docker image from a Dockerfile in the current directory.

Replace IMAGE_NAME with your desired image name.

docker pull IMAGE_NAME[:TAG]

Pulls a Docker image from a registry (like Docker Hub).

Example: docker pull ubuntu:latest

docker push USERNAME/IMAGE_NAME[:TAG]

Pushes a Docker image to a registry.

You need to be logged in to the registry using docker login.

docker images

Lists locally available Docker images.

docker rmi IMAGE_ID

Removes a Docker image.

You can get the IMAGE_ID using docker images.

Other Useful Commands

docker-compose up -d

Starts services defined in a Docker Compose file in detached mode.

docker-compose down

Stops and removes containers, networks, volumes, and images created by docker-compose up.

docker logs CONTAINER_ID

View the logs of a Docker container.

This is a basic overview of Jenkins and Docker with some common commands. There are many more features and options available for both tools.
